# Power-BI-Sales-Analytics-Project
End-to-end Sales Analytics dashboard built using Power BI and SQL Server. Features interactive visualizations, KPI tracking, sales trends, product analysis, customer insights, and geographic performance analysis.
## 🛠️ Tools & Technologies

- Power BI
- SQL Server
- Microsoft Excel

## 📈 Dashboard Features

- Sales Performance Dashboard
- Sales Trend Analysis
- Product Analysis
- Geographic Sales Analysis
- Customer Analysis
- Interactive KPIs & Filters
